# 🎙 Диктовка

Локальная голосовая диктовка для Mac. Говоришь — текст появляется там, где курсор:
в браузере, заметках, мессенджере, где угодно.

- **Локально.** Распознавание идёт на твоём Mac. Без облаков, без API, без подписок.
- **Приватно.** Аудио живёт только в памяти и в сеть не уходит. На диск пишется
  только распознанный текст — в твой личный файл `~/Documents/Диктовка.txt`.
- **Ничего не теряется.** Речь распознаётся по ходу (на паузах) и каждая фраза
  сразу пишется в этот файл. Даже если приложение закрыть во время записи —
  всё уже сказанное останется в файле.
- **Два способа набора** на правый Option:
  - **держи и говори** (отпустил — вставилось);
  - **двойной тап** → говоришь без рук → **одиночный тап** → вставилось
    (hands-free сам остановится через 5 минут, чтобы микрофон не висел).
- Русский и английский (определяется сам).

После установки сворачивается в значок 🎙 в строке меню и сама запускается при
включении компьютера. Думать о ней не нужно.

---

## Что нужно

- **Mac на Apple Silicon** (M1/M2/M3/M4). На старых Intel-маках не работает.
- **Python 3.10+**. Проверь в Терминале: `python3 --version`.
  Если нет или старый — поставь с [python.org](https://www.python.org/downloads/macos/)
  (большая кнопка Download) и открой установщик.

## Установка

1. Открой приложение **Терминал** (Spotlight: ⌘+Пробел → набери «Терминал»).
2. Скопируй и вставь туда **одну** строку, нажми Enter:

   ```bash
   git clone https://github.com/DenisRang/mac-dictation.git && cd mac-dictation && ./install.sh
   ```

   Установка скачает ~300 МБ и займёт пару минут.

3. Когда увидишь «Готово!» — осталось **выдать 3 разрешения** (один раз).
   Открой **System Settings → Privacy & Security** и включи галку у **«Python»**
   в трёх разделах:

   | Раздел | Зачем |
   |---|---|
   | **Microphone** | слышать тебя |
   | **Accessibility** | вставлять текст |
   | **Input Monitoring** | ловить нажатие правого Option |

   > Пункт называется **«Python»** — это и есть Диктовка (так устроен запуск).

4. После включения **Accessibility** и **Input Monitoring** перезапусти приложение:
   значок 🎙 в строке меню → **«Выход»**, потом снова открой через Launchpad
   (набери «Диктовка»).

Готово. **Первая** диктовка один раз скачает модель распознавания (~1.5 ГБ) —
нужен интернет. Дальше всё работает офлайн.

## Как пользоваться

В любом текстовом поле:

- **Зажми правый Option**, говори, **отпусти** — текст вставится.
- Или **дважды быстро нажми** правый Option → говори → **нажми один раз** — вставится.

Значок 🎙 в строке меню: статус, смена языка (авто / ru / en), «Открыть лог
расшифровок» и выход.

Всё, что ты надиктовал, дублируется в файл **`~/Documents/Диктовка.txt`** (с
датой и временем) — на случай если текст вставился не туда или ты хочешь найти
сказанное позже.

## Удаление

В папке проекта:

```bash
./uninstall.sh
```

## Частые вопросы

**Ничего не вставляется.** Проверь, что включены все три разрешения у «Python»
(особенно **Accessibility**), и перезапусти приложение.

**Не реагирует на правый Option.** Включи **Input Monitoring** у «Python» и
перезапусти.

**Где значок?** Справа в строке меню (🎙Дикт). На маках с «чёлкой» он может
прятаться за вырезом — но для работы видеть его не обязательно.

**Почему «Python» в разрешениях, а не «Диктовка»?** Приложение запускает Python,
и macOS подписывает разрешения по нему. На работу не влияет.

## Как это устроено

Тонкая обёртка-`.app` запускает `dictation.py` из локального окружения
(`~/Library/Application Support/Диктовка/venv`). Распознавание — через
[mlx-whisper](https://github.com/ml-explore/mlx-examples) (Apple MLX). Никакого
сетевого кода в приложении нет; единственное обращение в сеть — разовая загрузка
модели с Hugging Face при первом запуске.
